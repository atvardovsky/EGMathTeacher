# AI Infrastructure Adaptation Flow

Compatibility alias for `.ai/assistant/flows/skill-adaptation.flow.md`.

Use `skill-adaptation.flow.md` for new work. This file remains so older
EGMathTeacher help text, reports, or user requests that mention "AI
infrastructure adaptation" still route safely.

Before importing anything:

1. Read `.ai/framework/skill-adaptation.md`.
2. Read `.ai/framework/prompt-injection.md`.
3. Read `.ai/assistant/policies/ai-infrastructure-source-access.md`.
4. Read `.ai/assistant/policies/prompt-injection.md`.
5. Run or reuse a current AI infrastructure inventory.
6. Continue with `.ai/assistant/flows/skill-adaptation.flow.md`.

Do not import, execute, install, or normalize external infrastructure from this
compatibility flow alone.
